About

This is a project for the course "Full Stack Software Development" at Code Institute. The goal of this project is to create a web application that helps users to manage their waste. The application will let the users know when the next waste collection is due. They can either follow an existing waste collection schedule or create their own schedule.

The code for this project is open source. Check out the Github repo.
